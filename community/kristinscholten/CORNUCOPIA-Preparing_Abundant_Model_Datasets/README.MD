## Overview

Integrates Solar-Induced Fluorescence (SIF) data (2015â€“2020) with USDA crop yields to create machine learning training datasets. Calculates crop-specific SIF ratios, areas (measured in km and acres), and bushels using satellite and yield data processed with DuckDB.
